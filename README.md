## PosterCraft AI

PosterCraft AI transforms raw product photos into campaign-ready advertising posters. Upload an asset, choose a creative style, and let the assistant remix copy, brand highlights, and layout in real time. Built with Next.js 16, React 19, and Tailwind CSS 4 for seamless deployment on Vercel.

### Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to use the studio.

### Key Features

- Live canvas renderer that composes gradients, product render, copy, CTAs, and highlight chips into a 4:5 poster.
- AI-aided copy refresh button that drafts taglines, supporting copy, and benefit bullets tuned to the selected art direction.
- Multiple creative directions (Launch Neon, Minimal Glow, Organic Fresh) with unique palettes, textures, and layout heuristics.
- Product image upload with instant preview plus a sample asset for quick experimentation.
- One-click PNG export for sharing across channels.

### Scripts

```bash
npm run dev     # local development
npm run build   # production build
npm run start   # run built app

```
